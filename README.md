# Python-Bank-Project
My first college Python project to create a simple terminal only bank program



## Getting Started
Use your prefered IDE or any free online python IDE and include the **main.py** file and **txt** file
Run the program!

## Features
- Sign up/Log in
- Account info
- Change PIN
- Withdraw
- Deposit
- Bills
- Transactions
- Terminate
